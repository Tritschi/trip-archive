# Roadmap

## v0.3.x — Recorder archive foundation

- permanent immutable GPS and odometer storage
- rebuildable derived statistics and display routes
- Recorder-only trip generation
- sidebar dashboard with interactive map
- stable browser and Companion App navigation

## v0.4.0 — Validation and diagnostics

- compare regenerated trips with reference archives
- archive integrity checks
- clear diagnostics for missing or incomplete trip files
- rebuild derived files from immutable raw data
- improved error reporting and recovery

## v0.5.0 — Trip structure

- daily stages
- stops and time gaps
- start, destination and stage markers
- rebuildable summaries and stage statistics

## v0.6.0 — Additional sensor streams

- fuel and consumption
- range and battery
- arbitrary Recorder sensor streams
- configurable statistics per source

## v1.0.0 — Stable

- migration guarantees
- backup and export tooling
- complete user and developer documentation
- HACS-ready packaging
